= Spring Boot Camel AMQP Producer

. Build the project
+
....
 mvn clean package
....

. Start the fat jar
+
....
 mvn spring-boot:run
....

- To run with a unique client ID
+
....
CLIENT_ID=producer_01 mvn spring-boot:run
....

- To change the destination address
+
....
DESTINATION=queue:someQ mvn spring-boot:run
....
